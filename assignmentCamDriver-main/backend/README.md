# 🚗 Fleet Monitoring Dashboard

A **real-time fleet monitoring dashboard** that simulates dashcam events and displays vehicle activity live.

---

## 📌 Features

- 🚘 Real-time vehicle monitoring
- 📹 Dashcam event simulation
- 🔌 WebSocket based live updates
- 📊 Interactive dashboard UI
- ⚡ Node.js backend API
- 🌐 Simple frontend interface

---

## 🛠 Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js
- WebSockets
